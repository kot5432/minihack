# MiniHack

**Minimal Focused Development Environment**

MiniHackは「テーマを決める → 集中する → 振り返る」の3ステップで完了する、最小限の集中作業ツールです。

## 🎯 特徴

- **25分タイマー**: 絶対時間計算で正確なカウントダウン
- **AI要約**: OpenAI APIで2つの要約候補を生成
- **セッション復元**: リロードしても状態を保持
- **レスポンシブ**: モバイル・デスクトップ対応
- **KPI収集**: 改善のためのデータ基盤

## 🚀 使い方

1. **テーマ設定**: ホーム画面で今日の作業テーマを入力
2. **集中作業**: 25分のタイマーで集中して作業
3. **振り返り**: 作業内容を記録し、AIで要約

## 🛠️ 技術スタック

- **フロントエンド**: Next.js 14 + TypeScript + Tailwind CSS
- **AI連携**: OpenAI GPT-3.5-turbo
- **状態管理**: localStorage + React Hooks
- **デプロイ**: Vercel対応

## 📊 MVPとしての完成度
